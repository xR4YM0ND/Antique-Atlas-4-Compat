This resourcepack adds modded support for [aa4](https://modrinth.com/mod/antique-atlas-4) & various world generation / structure mods listed below.

## Supported Mods:
</center>

- [[Let's Do] BloomingNature](https://modrinth.com/mod/lets-do-bloomingnature) `// biomes`
- [Ad Astra](https://modrinth.com/mod/ad-astra) `// biomes`
- [Atmospheric](https://modrinth.com/mod/atmospheric) `// biomes`
- [Autumnity](https://modrinth.com/mod/autumnity) `// biomes`
- [Biomes O Plenty](https://modrinth.com/mod/biomes-o-plenty) `// biomes`
- [Blooming Biosphere](https://modrinth.com/mod/blooming-biosphere) `// biomes`
- [Cliff under a Tree (CliffTree)](https://modrinth.com/datapack/clifftree) `// biomes`
- [Environmental](https://modrinth.com/mod/environmental) `// biomes`
- [Nature's Spirit](https://modrinth.com/mod/natures-spirit) `// biomes + structures + markers`
- [Oh The Biomes We've Gone](https://modrinth.com/mod/oh-the-biomes-weve-gone) `// biomes`
- [Spawn](https://modrinth.com/mod/spawn-mod) `// biome`
- [Terralith](https://modrinth.com/datapack/terralith) `// biomes`
- [Vanilla Backport](https://modrinth.com/mod/vanillabackport) `pale_oak biome`

<center>

If you want support for other mods let me know.
